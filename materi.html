Pengenalan HTML

HTML (HyperText Markup Language) bukanlah bahasa pemrograman, melainkan bahasa markup. Tugasnya adalah memberi tahu browser bagaimana menyusun konten sebuah halaman. Bayangkan HTML sebagai kerangka (skeleton) dari tubuh manusia; ia merupakan struktur dasarnya.

Anatomi Tag HTML

Elemen HTML biasanya terdiri dari tag pembuka, konten, dan tag penutup. Beberapa tag tidak memiliki tag penutup, yang disebut empty tags (contoh: <img>, <br>, <input>).

<p>Ini adalah konten paragraf.</p>

Banyak tag juga memiliki atribut, yang memberikan informasi tambahan tentang elemen tersebut. Atribut selalu ada di tag pembuka.

<a href="https://google.com">Ini link ke Google</a>

Struktur Dasar Dokumen

Setiap file HTML wajib memiliki struktur dasar ini untuk dianggap valid oleh browser.

<!DOCTYPE html>
<html>
<head>
    <!-- Informasi untuk browser (tidak terlihat) -->
    <title>Judul Halaman di Tab Browser</title>
</head>
<body>
    <!-- Semua konten yang terlihat oleh pengguna -->
    <h1>Judul Utama</h1>
    <p>Sebuah paragraf.</p>
</body>
</html>
  • <!DOCTYPE html>: Mendefinisikan bahwa dokumen ini adalah HTML5.
  • <html>: Elemen utama yang membungkus semua konten halaman.
  • <head>: Berisi meta-informasi seperti judul halaman, link ke file CSS, dan lainnya.
  • <body>: Berisi semua konten yang akan ditampilkan di halaman web.

Elemen Penting: Block vs. Inline

Elemen HTML secara umum terbagi menjadi dua tipe tampilan:

  • Block-level Element: Selalu dimulai di baris baru dan memakan seluruh lebar yang tersedia. Contoh: <h1>, <p>, <div>.
  • Inline-level Element: Tidak dimulai di baris baru dan hanya memakan lebar sesuai kontennya. Contoh: <a>, <span>, <strong>.

<div> (block) adalah "pembungkus" generik, sedangkan <span> (inline) digunakan untuk mengelompokkan sebagian kecil teks dalam satu baris.

Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa yang kita gunakan untuk "menghias" kerangka HTML. Jika HTML adalah kerangka, maka CSS adalah pakaian dan gaya rambutnya. CSS mengontrol semua yang berhubungan dengan visual.

3 Cara Menambahkan CSS

  1. External CSS (Cara Terbaik): Menulis CSS di file terpisah (.css) dan menghubungkannya di dalam <head>. Ini cara paling rapi.
    2. <link rel="stylesheet" href="style.css">
  2. Internal CSS: Menulis CSS di dalam tag <style> di dalam <head>.
  3. Inline CSS: Menambahkan gaya langsung ke tag HTML menggunakan atribut style. Sebaiknya dihindari.

Anatomi Aturan CSS

Sebuah aturan CSS terdiri dari selektor dan blok deklarasi.

/* p adalah Selektor */
p { 
    /* color: blue; adalah Deklarasi */
    color: blue;      /* 'color' adalah Properti, 'blue' adalah Nilai */
    font-size: 16px; 
}

Konsep Kunci: Box Model

Setiap elemen HTML dapat dianggap sebagai sebuah kotak. CSS Box Model adalah aturan yang mendefinisikan ukuran dan jarak kotak tersebut. Ini sangat penting untuk memahami tata letak.

  • Content: Isi dari kotak (teks, gambar).
  • Padding: Ruang transparan di dalam kotak, antara konten dan border.
  • Border: Garis yang mengelilingi padding dan konten.
  • Margin: Ruang transparan di luar kotak, yang mendorong elemen lain menjauh.
.kotak-spesial {
    width: 200px;
    padding: 20px; /* Jarak di dalam 20px */
    border: 2px solid black; /* Garis batas 2px */
    margin: 30px; /* Jarak di luar 30px */
}
